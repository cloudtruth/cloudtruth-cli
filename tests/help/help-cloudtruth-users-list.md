```console
$ cloudtruth users list --help
cloudtruth-users-list 
List CloudTruth users

USAGE:
    cloudtruth users list [FLAGS] [OPTIONS]

FLAGS:
    -h, --help          Prints help information
        --show-times    Show create and modified times.
    -v, --values        Display account information/values
    -V, --version       Prints version information

OPTIONS:
    -f, --format <format>    Format for account values data [default: table]  [possible values: table, csv, json, yaml]
```
