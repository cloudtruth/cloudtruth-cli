$ cloudtruth parameter-types list --help
cloudtruth-parameter-types-list 
List CloudTruth parameter types

USAGE:
    cloudtruth parameter-types list [FLAGS] [OPTIONS]

FLAGS:
    -h, --help          Prints help information
        --rules         Display the parameter type rules.
        --show-times    Show create and modified times.
    -v, --values        Display parameter type information/values
    -V, --version       Prints version information

OPTIONS:
    -f, --format <format>    Display parameter type value info format [default: table]  [possible values: table, csv,
                             json, yaml]
