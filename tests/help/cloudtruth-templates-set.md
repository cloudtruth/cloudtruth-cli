$ cloudtruth templates set --help
cloudtruth[EXE]-templates-set 
Set the CloudTruth template

USAGE:
    cloudtruth[EXE] templates set [OPTIONS] <NAME>

FLAGS:
    -h, --help       Prints help information
    -V, --version    Prints version information

OPTIONS:
    -b, --body <FILE>           File containing the template
    -d, --desc <description>    Template description
    -r, --rename <new-name>     New template name

ARGS:
    <NAME>    Template name
