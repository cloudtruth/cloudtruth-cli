$ cloudtruth configuration profiles list --help
cloudtruth[EXE]-configuration-profiles-list 
List CLI profiles

USAGE:
    cloudtruth[EXE] configuration profiles list [FLAGS] [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -s, --secrets    Display API key values
    -v, --values     Display profile information/values
    -V, --version    Prints version information

OPTIONS:
    -f, --format <format>    Display profile value info format [default: table]  [possible values: table, csv, json,
                             yaml]
