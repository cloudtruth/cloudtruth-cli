$ cloudtruth environments tag list --help
cloudtruth-environments-tag-list 
List CloudTruth environment tags

USAGE:
    cloudtruth environments tag list [FLAGS] [OPTIONS] <env-name>

FLAGS:
    -h, --help       Prints help information
    -v, --values     Display environment tag information
    -V, --version    Prints version information

OPTIONS:
    -f, --format <format>    Format for environment tag values data [default: table]  [possible values: table, csv,
                             json, yaml]

ARGS:
    <env-name>    Environment name
