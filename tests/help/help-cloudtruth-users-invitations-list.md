$ cloudtruth users invitations list --help
cloudtruth-users-invitations-list 
List user invitations

USAGE:
    cloudtruth users invitations list [FLAGS] [OPTIONS]

FLAGS:
    -h, --help       Prints help information
    -v, --values     Display invitation values
    -V, --version    Prints version information

OPTIONS:
    -f, --format <format>    Format for invitation values [default: table]  [possible values: table, csv, json, yaml]
