$ cloudtruth actions imports list --help
cloudtruth-actions-imports-list 
List CloudTruth imports

USAGE:
    cloudtruth actions imports list [FLAGS] [OPTIONS]

FLAGS:
    -h, --help          Prints help information
        --show-times    Show create and modified times.
    -v, --values        Show import info values
    -V, --version       Prints version information

OPTIONS:
    -f, --format <format>       Format for import info [default: table]  [possible values: table, csv, json, yaml]
    -i, --integration <name>    Integration name
