cloudtruth[EXE]-parameters-pushes 
Show push task steps for parameters

USAGE:
    cloudtruth[EXE] parameters pushes [FLAGS] [OPTIONS] [KEY]

FLAGS:
    -h, --help          Prints help information
        --show-times    Show create and modified times.
    -v, --values        Display push task step info
    -V, --version       Prints version information

OPTIONS:
    -f, --format <format>    Format for push task step info [default: table]  [possible values: table, csv, json, yaml]

ARGS:
    <KEY>    Parameter name
