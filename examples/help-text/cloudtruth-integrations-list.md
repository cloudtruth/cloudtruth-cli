```console
$ cloudtruth integrations list --help
cloudtruth[EXE]-integrations-list 
List CloudTruth integrations

USAGE:
    cloudtruth[EXE] integrations list [FLAGS] [OPTIONS]

FLAGS:
    -h, --help          Prints help information
        --show-times    Show create and modified times.
    -v, --values        Display integration information/values
    -V, --version       Prints version information

OPTIONS:
    -f, --format <format>    Format for integration values data [default: table]  [possible values: table, csv, json,
                             yaml]
```
